## Bootstrap

```bash
sh <(curl -s https://getconduit.dev/bootstrap)
```

## Deploy

```bash
conduit deploy setup --config
```

Or without global install:

```bash
npx @conduitplatform/cli deploy setup
```

See [Install](/docs/getting-started/install) for Docker alternatives.
