## Is Conduit open source?

Yes — Apache 2.0. [GitHub](https://github.com/ConduitPlatform/Conduit).

## Self-hosted vs managed?

Conduit is **self-hosted by default**. [Conduit Hosting](/hosting) (managed instances) is coming soon.

## Which API should my app use?

**Client API** only (`:3000`) with user bearer tokens. See [Client vs Admin API](/docs/learn/client-vs-admin-api).

## How do AI agents provision schemas?

Configure the [Conduit MCP](/docs/getting-started/mcp-setup) with admin credentials. Use [Start with AI](/docs/getting-started/with-llms) for a copy-paste prompt.

## Where is email/SMS/push documented?

In v0.17, use the unified [Communications module](/docs/modules/communications).

## What Node version for development?

**Node.js 24+** and **pnpm** for the Conduit monorepo.
