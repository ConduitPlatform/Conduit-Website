## Development setup

1. Clone [ConduitPlatform/Conduit](https://github.com/ConduitPlatform/Conduit)
2. **Node.js 24+** and **pnpm**
3. `pnpm install --frozen-lockfile`
4. `turbo run build`
5. Start core, then modules — see [Local from source](/docs/deployment/local-source)

## Guidelines

- Follow conventional commits
- Use pnpm only (not npm/yarn) in the monorepo
- Client API vs Admin API boundaries apply to all example code in PRs
