| Version | Location |
|---------|----------|
| **v0.17** (current) | [getconduit.dev/docs/intro](/docs/intro) — this site |
| **v0.16** | [archive.getconduit.dev/v0.16](https://archive.getconduit.dev/v0.16) |

URLs like `/docs/v0.16/modules/database/get-started` on the main domain redirect to the archive site automatically.

## When to use legacy docs

- You are still running **v0.15** or **v0.16** and need module-specific tutorials with admin panel screenshots
- You need historical reference for **email**, **sms**, or **push-notifications** as separate modules (consolidated into **communications** in v0.17)

For upgrading, start with [Migration v0.16 → v0.17](/docs/resources/migration-v0.16-to-v0.17) and the [changelog](/docs/resources/changelog).
