# GraphQL API

## Performing Requests

Administrative GraphQL routes require that you specify certain [administration headers](./#headers).<br />
Your administrative GraphQL path is `/graphql`.<br />
You may always retrieve the exact url through the Admin Panel.

<div style={{textAlign: 'center'}}>

![GraphQL Button](/img/overview/home_topright_buttons.png)

</div>

Navigating to the above URL through your web browser should bring up GraphQL Playground.<br />
You can use that to test out your requests.
