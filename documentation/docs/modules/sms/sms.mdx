# SMS

## Overview
Sms module is used mainly to support Two Factor Authentication.<br />
In order to start experiment with this module, you have to create a twilio account. Sign-up [here](https://www.twilio.com/try-twilio).<br />
Since you have configured the provider you will be able to "play" with this module.<br />
Find out more about configuring the sms provider [here](./config)
