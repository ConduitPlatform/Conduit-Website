# Config
Since you've created an account , you need some configuration to proceed with the activation of the module.

## Mailgun
```json
MailgunConfig {
  auth: {s
    api_key: string;
    domain: string;
  };
  proxy?: string;
  host: string;
}
```
### Api Key
[See here how to find your api key that is linked with your account.](https://help.mailgun.com/hc/en-us/articles/203380100-Where-Can-I-Find-My-API-Key-and-SMTP-Credentials-)
