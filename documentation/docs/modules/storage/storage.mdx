# Storage

![Storage Module](/img/modules/storage/admin/storage_module.png)

This module provides persistent storage and facilitates the creation of hierarchical data structures.<br />
Documents may have complex structures and subcollections and can be stored locally or on a cloud provider.


## Features

- Local storage
- Microsoft Azure
- Google Cloud
