# CMS module

![Schema View](/img/modules/cms/admin/custom_endpoint_edit_view.png)

This module facilitates the management of content types through the generation 
of db level schemas, the creation and manipulation of schema document entries 
and the registration of fine-tuned custom endpoints.

Content is delivered through REST and GraphQl.

## Features

- Create any content type (schema)
- Create db collections based on said types
- Create db collection entries
- Create powerful custom endpoints
- Expose content through GraphQL
- Expose content through REST API
- Swagger UI (OpenAPI)
- Pagination
- Sorting
- GraphQl-style 'populate' field for REST APIs
