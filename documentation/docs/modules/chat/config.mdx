# Configuration

Bring up the admin panel and head to the module's administration page, 
selecting the `Config` tab.

## Available Options

![Configuration View](/img/modules/chat/admin/config_view.png)

### Allow Message Edit

Enables support for chat message modification.

### Allow Message Delete

Enables support for chat messages deletion.

## Updating the Module's Config

Edit the configuration and save your changes.

![Configuration Edit](/img/modules/chat/admin/config_edit.png)

That's it. Your configuration is now updated.
