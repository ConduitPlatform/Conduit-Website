# Configuration

This module is automatically enabled.<br />
Messages can by modified or deleted by default.

## Available Options

|   Option           |   Description               | Default |
| :----------------: | :-------------------------: | :-----: |
| active             | Enables the module          |  True   |
| allowMessageDelete | Enables Message Deletion    |  True   |
| allowMessageEdit   | Enables Message Modifiation |  True   |

## Updating the Module's Config

Bring up the admin panel and head to the Chat module's administration page.<br />
Select the "Config" tab.

![Configuration View](/img/modules/chat/admin/config_view.png)

Edit the configuration and save your changes.

![Configuration Edit](/img/modules/chat/admin/config_edit.png)

That's it. Chat is now following your new configuration.
