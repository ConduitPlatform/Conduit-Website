# Custom Endpoint User Routes

Custom endpoints are exposed under `/database/function/:customEndpointName`.

<ins><strong>Notice:</strong> Routes are <strong>only</strong> generated for <strong>enabled</strong> custom endpoints, whose schema also <strong>enables <em>crudOperations</em></strong>.</ins>

## Parameters

A Custom endpoint's parameters are defined during CustomEndpoint creation.

Paginated **GET** routes enable support for the following url parameters:

| URL params |  Type   | Required |  Description              | Example |
| :--------: | :-----: |:-------: | : ----------------------: | :-----: |
| skip       | Integer | True     | Docs to skip              | 20      |
| limit      | Integer | True     | Docs to get               | 20      |
| sortObj    | String  | False    | Sort by fields            |   **TODO**      |
| populate   | String  | False    | Relation fields to expand |   **TODO**      |
