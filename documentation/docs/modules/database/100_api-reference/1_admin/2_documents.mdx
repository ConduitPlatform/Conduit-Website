# Schema Document Admin Routes

## Get Document

|   Route                                      | Method |
| :------------------------------------------: | :----: |
| /admin/database/schemas/:schemaName/docs/:id |  GET   |

Get a schema document by id.

### Url parameters

- schemaName (string)
- id (string)

### Returns

The document in JSON format.

```json title="Example Response"
{
  "result": {
    "_id": "61e7debc5dadc501b101d166",
    "firstName": "Nick",
    "lastName": "Brown",
    "ssn": "AAA-14-6823",
    "age": 32,
    "gender": "M",
    "createdAt": "2022-01-19T09:16:33.827Z",
    "updatedAt": "2022-01-19T09:16:33.827Z",
    "__v": 0
  },
}
```

## Get Documents

|   Route                                   | Method |
| :---------------------------------------: | :----: |
| /admin/database/schemas/:schemaName/query |  POST  |

Get schemas documents.

### Url parameters

- schemaName (string)

### Query parameters

|     Name    |  Type   | Required | Description |
| :---------: | :-----: | :------: | :---------: |
| skip        | Integer |  False   |             |
| limit       | Integer |  False   |             |

### Body parameters

|     Name    |  Type   | Required | Description |
| :---------: | :-----: | :------: | :---------: |
| query       | JSON    |  True    |             |


### Returns

- documents (JSON[])
- count (integer)

## Update Document

|   Route                                      | Method  |
| :------------------------------------------: | :-----: |
| /admin/database/schemas/:schemaName/docs/:id |  PUT    |

Update a schema document by id.
The document is completely replaced and any missing optional fields are nulled.

### Url parameters

- schemaName (string)
- id (string)

### Body parameters

|     Name        |  Type   | Required | Description |
| :-------------: | :-----: | :------: | :---------: |
| changedDocument | JSON    |  True    |             |


### Returns

The document in JSON format.
