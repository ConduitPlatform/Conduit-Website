# Authentication

![Authentication Module](/img/modules/authentication/admin/overview.png)

This module offers user authentication.<br />
It supports logins with local credentials as well as popular identity providers.<br />
TwoFactor authentication is also provided.

## Features

- Local authentication with username/password or email/password (requires [Email](../email/email.mdx) module)
    - Optional email verification on email/password strategy
    - Forgot password
    - Reset password
- Facebook Login with client credentials
- Google Login with client credentials
- Kakao Talk login with redirect strategy
- Twitch login with redirect strategy
- Service account / API key
- Optional 2FA supported on all strategies (requires [SMS](../sms/sms.mdx) module)
- Optional Account linking supported when email is the same between providers
