# gRPC Service
## Set Configuration
|        RPC        | GrpcRequest | GrpcResponse
| :-----------------:    | :----: | :----:|
| setConfig |     SetConfigRequest | SetConfigResponse

```proto
message SetConfigRequest {
  string newConfig = 1;
}

message SetConfigResponse {
  string updatedConfig = 1;
}
```

Sets the configuration of the authentication module

## User Login
|        RPC        | GrpcRequest | GrpcResponse
| :-----------------:    | :----: | :----:|
| userLogin |     UserLoginRequest | UserLoginResponse

```proto
message UserLoginRequest {
  string userId = 1;
  string clientId = 2;
}

message UserLoginResponse {
  string accessToken = 1;
  string refreshToken = 2;
}
```
Login a user.

## User Creation

|        RPC        | GrpcRequest | GrpcResponse
| :-----------------:    | :----: | :----:|
| userCreate |     UserCreateRequest | UserCreateResponse
```proto
message UserCreateRequest {
  string email = 1;
  optional string password = 2;
  bool verify = 3;
}

message UserCreateResponse {
  optional string password = 1;
}
```
Used for user creation.
## Change User Password
|        RPC        | GrpcRequest | GrpcResponse
| :-----------------:    | :----: | :----:|
| changePass |     UserChangePass | UserCreateResponse
```proto
message UserChangePass {
  string email = 1;
  optional string password = 2;
}

message UserCreateResponse {
  optional string password = 1;
}
```
RPC call for changing user's password.
