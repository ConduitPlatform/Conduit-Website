## Swagger

Conduit's Router automatically generates OpenAPI schemas for your routes to be by Swagger.<br />
This means your REST API is fully documented and even exposed through Swagger UI.<br />
Check out your own [User](http://localhost:3000/swagger/) and [Admin](http://localhost:3000/admin/swagger/) Swagger UI pages.

Oh, and you can grab your own [User](http://localhost:3000/swagger.json) and [Admin](http://localhost:3000/admin/swagger.json) *swagger.json* files 
and import them directly in any API testing app of your choice<br />

Just make sure you set up all of the [required headers](#headers) and point your `baseUrl` variable to the right path.<br />
