# Conduit gRPC SDK

This library provides a framework to bind all conduit modules in a safe way.<br />
It is also used to register and replace modules.

In the future, this could be used to entirely replace modules like the CMS or even the database provider.

## Build Steps

- run build.sh
- run npm build

## Features

- Defines an Interface for all modular functionalities of Conduit
- Provides constructors for model declaration in Conduit
