# Config
Since you've created an account on one of the [Supported Providers](./#supported-providers), you need to configure the provider to proceed with the activation of the module.

## Mailgun
### Sending Domain
See here how to [Verify Your Domain](https://documentation.mailgun.com/en/latest/user_manual.html#verifying-your-domain-1).
### Api Key
See here how to [Find Your Api Key](https://help.mailgun.com/hc/en-us/articles/203380100-Where-Can-I-Find-My-API-Key-and-SMTP-Credentials-) that is linked with your account.
### Host
See here for mailgun [host](https://documentation.mailgun.com/en/latest/api-intro.html#introduction) documentation.

![Provider Configuration](/v0.14/modules/email/admin/provider_configuration.png)

## Mandrill
All you need to start sending e-mails through mandrill provider is the Api Key. <br/>
### Api Key
Since you are logged in your mandrill account:

- Go to Settings in your Mandrill account. <br />
- Click + New API Key.<br />
- Enter a description of the key.<br />
- Click Create API Key.<br />

![Mandrill Configuration](/v0.14/modules/email/admin/mandrill_configuration.png)


## Sendgrid
Only Api Key is required. <br />
See how to [Manage Sendgrid Api Keys](https://docs.sendgrid.com/ui/account-and-settings/api-keys).
