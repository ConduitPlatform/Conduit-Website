# Configuration

Bring up the admin panel and head to the module's administration page, 
selecting the `Config` tab.

## Available Options

![Configuration View](/v0.14/modules/forms/admin/config_view.png)

### Use Attachments

Enables support for attachment fields in forms.

<ins>Requirement:</ins>

Depends on the [Storage](../storage/storage.mdx) module being available.

## Updating the Module's Config

Edit the configuration and save your changes.

![Configuration Edit](/v0.14/modules/forms/admin/config_edit.png)

That's it. Your configuration is now updated.
