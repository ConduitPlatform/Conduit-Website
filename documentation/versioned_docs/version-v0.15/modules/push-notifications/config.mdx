# Configuration

In order to start sending notifications you need to create a Firebase project.<br />
To configure the module, all you need is three project secrets.

## Firebase Project Secrets

![Configuration View](/v0.15/modules/push-notifications/admin/config.png)

- Project Id
- Private Key
- Client Email

See more about project credentials [here](https://firebase.google.com/docs/admin/setup#add_firebase_to_your_app).

## Updating the Module's Config

Edit the configuration and save your changes.

That's it. Your configuration is now updated.
