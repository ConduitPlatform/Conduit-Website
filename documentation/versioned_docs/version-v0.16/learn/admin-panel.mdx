The Admin Panel is the command center for managing your Conduit deployment.

![Admin Panel Login](/v0.16/ui/login.png)

**Default credentials:** `admin` / `admin`

## Overview

From the Admin Panel, you can:

- Create and manage database schemas
- Configure custom endpoints
- Set up authentication methods
- View logs and metrics
- Administer users
- Configure module settings

![Admin Panel Home](/v0.16/ui/home.png)

## Navigation

The left navigation pane displays:

- **Available modules** - Online modules with their features
- **Offline modules** - Listed at the bottom
- **Settings** - Core configuration options
- **Log out** - Session management

## Module Management

Each module has its own dedicated page accessible from the navigation:

![Database Schema View](/v0.16/modules/database/admin/schema_list_view.png)

### Configuration

Configurable modules provide a **Settings** tab where you can:

- View current configuration
- Update settings
- Enable/disable features

For module-specific administration details, see the [Modules documentation](../modules/overview).

## Documentation Access

Click the **Documentation** button in the top-right corner of your homepage to quickly access these docs.
